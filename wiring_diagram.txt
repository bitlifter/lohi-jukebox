Jukebox Escape Room - Wiring Diagram
====================================

This is a text-based representation of how to wire the 1185RE8 microswitches to the Raspberry Pi Zero 2 W.

---------------------------------
                    ┌───┐
               3.3V │1 2│ 5V
          (GPIO2) SDA │3 4│ 5V
          (GPIO3) SCL │5 6│ GND
          (GPIO4) GP4 │7 8│ GPIO14 (TXD)
                  GND │9 10│ GPIO15 (RXD)
BUTTON1 → (GPIO17) GP17 │11 12│ GPIO18 (PCM_CLK)
BUTTON2 → (GPIO27) GP27 │13 14│ GND
BUTTON3 → (GPIO22) GP22 │15 16│ GPIO23
                 3.3V │17 18│ GPIO24
          (GPIO10) MOSI │19 20│ GND
           (GPIO9) MISO │21 22│ GPIO25
          (GPIO11) SCLK │23 24│ GPIO8 (CE0)
                  GND │25 26│ GPIO7 (CE1)
           (GPIO0) ID_SD │27 28│ GPIO1 (ID_SC)
           (GPIO5) GP5 │29 30│ GND
BUTTON4 → (GPIO6) GP6 │31 32│ GPIO12
RESET  → (GPIO13) GP13 │33 34│ GND
          (GPIO19) PWM1 │35 36│ GPIO16
          (GPIO26) GP26 │37 38│ GPIO20
                  GND │39 40│ GPIO21
                    └───┘

Note: This diagram uses BCM pin numbering. The physical pin numbers are shown in the diagram.

Microswitch Wiring (for each button)
-----------------------------------

    Raspberry Pi                   1185RE8 Microswitch
    -----------                    -------------------
                                        ┌───┐
    GPIO Pin (e.g., GPIO17) ────────────┤NO  │
                                        │    │
    GND Pin ───────────────────────────┤COM │
                                        └───┘

    * NO = Normally Open terminal
    * COM = Common terminal
    * NC = Normally Closed terminal (not used in this project)

Wiring Instructions
------------------

1. For each button, connect:
   - The COM terminal to a GND pin on the Raspberry Pi
   - The NO terminal to the appropriate GPIO pin

2. Recommended GPIO pins (BCM numbering):
   - Button 1: GPIO17 (Physical pin 11)
   - Button 2: GPIO27 (Physical pin 13)
   - Button 3: GPIO22 (Physical pin 15)
   - Button 4: GPIO5  (Physical pin 29)
   - Reset:    GPIO6  (Physical pin 31)

3. You can use any available GND pins on the Raspberry Pi. There are multiple GND pins
   available (pins 6, 9, 14, 20, 25, 30, 34, 39).

Notes
-----

1. The internal pull-up resistors on the Raspberry Pi will be enabled in software,
   so no external resistors are needed.

2. When a button is pressed, it will connect the GPIO pin to GND, pulling it LOW.
   The software is configured to detect this falling edge.

3. Make sure all connections are secure and that there are no short circuits.
